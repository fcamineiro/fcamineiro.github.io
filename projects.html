FIT File Dashboard

The FIT File Dashboard is a Flask-based web application designed to visualize and interact with fitness data stored in .FIT files. This project provides an intuitive interface for users to upload and analyze fitness data, displaying key metrics and trends through charts and summary tables.

MIMIC-III Dashboard

The MIMIC-III Dashboard is a dynamic web application built with Shiny for Python. It's designed to offer biostatistical analyses and visualization capabilities, focusing on data-driven insights. Through interactive charts and tables, users can deeply explore the MIMIC-III dataset.
